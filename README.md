# E-Commerce Laravel Project

It is a modern Laravel Blade–based eCommerce website designed for selling sports jerseys online. It features a clean, responsive frontend with product categories, search, cart, and smooth checkout (Cash on Delivery). The platform includes a powerful admin panel to manage categories, products with multiple images and sizes, orders, and shipping settings. Customers can view order history, receive email notifications, and download invoice PDFs—making it a complete, user-friendly eCommerce solution.
